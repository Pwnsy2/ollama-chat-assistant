 Ollama Chat Assistant

A simple Python CLI tool to chat with local AI models running via Ollama.

 What it does
Lets you have a conversation with any locally running Ollama model 
directly from your terminal. No internet required, no subscriptions, 
completely private.

 Requirements
- Python 3.x
- Ollama installed and running locally
- requests library (`pip install requests`)

 How to run
```bash
python ollama_chat.py
```

 Models tested
- Mistral
- Llama 3.1 70b
- Gemma 3 12b
